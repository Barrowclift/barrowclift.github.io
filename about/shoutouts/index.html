Shoutouts

My website is powered by Jekyll (the Ruby static website generator) and is hosted by Github Pages.

Code

I am genuinely grateful for all the open source projects and tips I utilized from the developer community, including:

  • BigFoot for their Instapaper-like footer popups use to power my article footnotes.
  • CSSStickyFooter for their wonderful sticky footer solution use for my copyright banner on the bottom of every page.
  • Hint for the lovely hover bubbles used in the Art and Experience pages.
  • Jacob Buck for his clever auto-updating copyright year script.
  • jQueryScript.net for their on-hover anchor links used for my article section headers.
  • Michael J. Deeb for his help with getting a custom domain set up with Github Pages and Hover.
  • mikeygee for sharing his clever solution for splitting up articles on the home page.
  • richleland on Github for his Monokai code syntax style which I use as a base for my code block color scheme.
  • SimpleForm for providing the contact form backend used on my Contact page.
